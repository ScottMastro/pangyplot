################
pangyplot status
################

Check database status.

SYNOPSIS
========

**pangyplot status** [*OPTION*]â€¦

DESCRIPTION
===========

Verifies and reports database contents.

OPTIONS
=======

FILTER OPTIONS
~~~~~~~~~~~~~~

| **--db** *NAME*
| Specific database name to check (default: all).

| **--dir** *DIR*
| Database directory (default: ``datastore``).

| **--table** *TABLE*
| Specific database table to show (e.g. ``segment``, ``link``, ``bubble``).
